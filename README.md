# inferno-redux-example

> An example of Inferno Redux async application

## Install

```sh
git clone https://github.com/andrepolischuk/inferno-redux-example
cd inferno-redux-example
npm install
```

## Run

```sh
npm start
```

## Test

```sh
npm test
```

## Build

```sh
npm run build
```
